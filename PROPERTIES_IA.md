# Properties Panel — Information Architecture

## Element
- Name
- Visibility toggle
- Tag (semantic HTML)

## Attributes (collapsed)
- Class Name
- HTML ID

## Content (per element type)
- **text**: content textarea, href (when tag=a)
- **image**: src, alt, object-fit
- **button**: content
- **input**: placeholder, type, disabled
- **textarea**: placeholder, rows, disabled
- **select**: options list, disabled
- **link**: content, href

## Position (collapsed)
- Position (static, relative, absolute, fixed, sticky)
- Z-Index
- Top
- Right
- Bottom
- Left

## Layout (box only)
- Display (block, flex, grid, inline-flex, inline-block)
- Direction (flex)
- Align / Justify (visual grid)
- Gap
- Wrap (flex)
- Grid Columns (grid)
- Grid Rows (grid)
- Overflow
- **Child behavior** (conditional: parent is flex or grid)
  - Grow
  - Shrink
  - Align Self
  - Col Span (grid)
  - Row Span (grid)

## Size
- Width
- Height
- Min Width
- Max Width
- Min Height
- Max Height

## Spacing
- Padding (4 sides)
- Margin (4 sides)

## Typography (not box, not image)
- Color
- Font Size
- Font Weight
- Line Height
- Letter Spacing
- Text Align
- Font Style (italic)
- Text Decoration (underline, strikethrough)
- Text Transform
- White Space

## Fill
- Background Color
- Opacity
- Background Image
- Background Size (when image set)
- Background Position (when image set)
- Background Repeat (when image set)

## Border
- Style
- Width
- Color
- Radius (4 corners)

## Effects
- Box Shadow
- Blur
- Backdrop Blur
- Cursor

## Transform (collapsed)
- Rotate
- Scale
- Translate X
- Translate Y

## Transition (collapsed)
- Transition property
- Duration
- Easing

## Classes (collapsed)
- Manual classes (editable pills)
- Computed classes (read-only)
